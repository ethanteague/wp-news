Fill your WordPress site with random users.

== Description ==

Randi is a random user importer for WordPress. Each user will get a first name, last name, email, username, registered date and photo.

[@randomapi](https://twitter.com/randomapi) is used to generate the user data.

== Installation ==

1. Go to Plugins > Add New and search for "randi"
2. Click on Install Now

You can then go ahead and activate the plugin.

== Frequently Asked Questions ==

= Is there a copyright notice I should be aware of? =

Yes. See [Copyright](https://randomuser.me/copyright).

= How can I generate random users? =

Go to Tools > Import and look for Randi in the list of importers.

= Why are user images broken when I use the plugin on my local machine? =

Randi uses Gravatar to serve up user images so images must be public. See [Image Requests](https://en.gravatar.com/site/implement/images/#default-image) for more info.

= Where should I submit bug reports? =

If you think you've spotted a bug, please open an issue on [GitHub](https://github.com/henrywright/randi/issues).
